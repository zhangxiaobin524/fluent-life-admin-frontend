# 音频文件说明

这个目录用于存放冥想背景音乐文件。

## 文件列表：

1. **ocean.mp3** - 海浪声
2. **drone.mp3** - 低频音
3. **white-noise.mp3** - 白噪音
4. **rain.mp3** - 雨声
5. **forest.mp3** - 森林声
6. **fire.mp3** - 火焰声

## 如何添加音频文件：

1. 将音频文件复制到此目录
2. 确保文件名与上述列表一致
3. 推荐使用 MP3 格式，兼容性最好
4. 文件大小建议控制在 1-5MB 内

## 在线备选：

如果本地文件不存在，系统会自动使用在线音频源作为备选。

## 注意事项：

- 音频文件会被浏览器缓存
- 建议使用无版权的音频素材
- 可以根据需要调整音频文件
